TI-99/4A
--------

Unify the individual C, D and G ROMS into a single ROM file you can load very simply.

Works with the TIMRaD rom set. Get the rom set here: http://www.99er.net/download2/index.php?act=category&id=21

./ti994a-unify-roms.clj --system-rom-path /tank/Downloads/BIOS_Files/ti994a/ROMs --rom-path /tank/Downloads/BIOS_Files/ti994a/MODULES --output output
